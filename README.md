- [ ] Lanzamiento de trabajos: foreground, background, suspensión...
- [ ] Lanzar comandos que produzcan fallos
- [ ] Comandos internos (bg, fg)
- [ ] Aniquilar o suspender jobs desde otro terminal (kill)
